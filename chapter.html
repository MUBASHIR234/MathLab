Quadratic Equations

Equations where the highest power of the variable is 2. This chapter covers solving them by factorization, completing the square, and the quadratic formula — plus word problems that turn into quadratics.

24 questions 4 exercises 1 quick test ~50 min

Concept Overview

A quadratic equation is any equation that can be written in the standard form below, where a ≠ 0.

ax² + bx + c = 0

There are three standard methods to solve it: factorization (splitting the middle term), completing the square, and the quadratic formula shown below.

x = (−b ± √(b² − 4ac)) / 2a

The expression under the root, b² − 4ac, is called the discriminant. It tells you whether the roots are real, equal, or complex — useful for checking your answer even before fully solving.

Exercise 1.1 — Solve by Factorization

6 questions
Q1Solve: x² − 5x + 6 = 0
HintFind two numbers that multiply to give 6 and add to give −5.
Solution
x² − 5x + 6 = 0
x² − 2x − 3x + 6 = 0
x(x − 2) − 3(x − 2) = 0
(x − 2)(x − 3) = 0
x = 2, x = 3
Q2Solve: 2x² + 7x + 3 = 0
HintMultiply a and c first (2 × 3 = 6). Find two numbers that multiply to 6 and add to 7.
Solution
2x² + 7x + 3 = 0
2x² + 6x + x + 3 = 0
2x(x + 3) + 1(x + 3) = 0
(x + 3)(2x + 1) = 0
x = −3, x = −1/2
Q3Solve: x² − 9 = 0
HintThis is a difference of squares: a² − b² = (a − b)(a + b).
Solution
x² − 9 = 0
(x − 3)(x + 3) = 0
x = 3, x = −3
Q4Solve: x² + 2x − 15 = 0
HintFind two numbers that multiply to −15 and add to 2.
Solution
x² + 2x − 15 = 0
x² + 5x − 3x − 15 = 0
x(x + 5) − 3(x + 5) = 0
(x + 5)(x − 3) = 0
x = −5, x = 3
Q5Solve: 3x² − 8x + 4 = 0
Hinta × c = 12. Find two numbers multiplying to 12, adding to −8.
Solution
3x² − 8x + 4 = 0
3x² − 6x − 2x + 4 = 0
3x(x − 2) − 2(x − 2) = 0
(x − 2)(3x − 2) = 0
x = 2, x = 2/3
Q6Solve: x² = 4x
HintBring all terms to one side first — don't divide by x, you'll lose a root.
Solution
x² − 4x = 0
x(x − 4) = 0
x = 0, x = 4

Exercise 1.2 — Completing the Square

4 questions
Q1Solve by completing the square: x² + 6x + 5 = 0
HintHalf of 6 is 3 — add and subtract 3² = 9.
Solution
x² + 6x + 5 = 0
x² + 6x + 9 − 9 + 5 = 0
(x + 3)² = 4
x + 3 = ±2
x = −1, x = −5
Q2Solve by completing the square: x² − 4x − 1 = 0
HintHalf of −4 is −2 — add and subtract (−2)² = 4.
Solution
x² − 4x − 1 = 0
(x − 2)² − 4 − 1 = 0
(x − 2)² = 5
x = 2 ± √5
Q3Solve: 2x² + 4x − 6 = 0 (complete the square)
HintDivide the whole equation by 2 first to make the leading coefficient 1.
Solution
x² + 2x − 3 = 0
(x + 1)² − 1 − 3 = 0
(x + 1)² = 4
x = 1, x = −3
Q4Solve: x² + 3x − 2 = 0 (complete the square)
HintHalf of 3 is 3/2 — your square term will involve a fraction.
Solution
(x + 3/2)² − 9/4 − 2 = 0
(x + 3/2)² = 17/4
x = (−3 ± √17) / 2

Exercise 1.3 — Quadratic Formula

3 questions
Q1Solve using the quadratic formula: x² − 3x − 1 = 0
HintIdentify a = 1, b = −3, c = −1, then substitute directly into the formula.
Solution
a = 1, b = −3, c = −1
x = (3 ± √(9 + 4)) / 2
x = (3 ± √13) / 2
Q2Solve: 2x² − 4x + 1 = 0
Hinta = 2, b = −4, c = 1. Simplify √8 = 2√2 at the end.
Solution
x = (4 ± √(16 − 8)) / 4
x = (4 ± 2√2) / 4
x = (2 ± √2) / 2
Q3Find the discriminant and nature of roots: x² + 4x + 4 = 0
HintDiscriminant = b² − 4ac. If it equals 0, roots are real and equal.
Solution
b² − 4ac = 16 − 16 = 0
Discriminant = 0 → roots are real and equal
x = −2, −2

Exercise 1.4 — Word Problems

2 questions
Q1The sum of a number and its square is 42. Find the number.
HintLet the number be x. Translate "sum of a number and its square" into x² + x.
Solution
x² + x = 42
x² + x − 42 = 0
(x + 7)(x − 6) = 0
x = 6 or x = −7
Q2A rectangle's length is 3 m more than its width. Its area is 70 m². Find its dimensions.
HintLet width = x, so length = x + 3. Area = length × width.
Solution
x(x + 3) = 70
x² + 3x − 70 = 0
(x + 10)(x − 7) = 0
Width = 7 m, Length = 10 m
